# Spooky_Monster_RU_Font_Patch

Патч для корректировки русского шрифта "Ужасный монстр".

У некоторых пользователей отмечается неправильный шрифт в русской версии.

Инструкция по исправлению ошибки:

1.Скачайте архив patch_ru_font.zip отсюда (зеленая кнопка CODE -> Download zip)

2.Распаковать архив

3.Установите шрифт CODE2000.TTF - надо нажать на него и всплывшем окне нажать "Установить"

4.Открыть файлы игры в Steam и VK Play:

-В Steam: Управление (кнопка-шестеренка) -> Посмотреть локальные файлы

-В VK Play: справа от кнопки "Играть" небольшая кнопка со стрелкой вниз  -> Открыть папку клиента игры

5.Переместить в корневую папку с игрой файл patch.xp3 

6.Можно играть
